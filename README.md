# N-Tool Golike TikTok

My first tool project: Golike TikTok Auto
Built by Me & AI Agent

## Features
* Uses ADB to connect from PC to phone (and Termux to phone).
* Automatically opens TikTok and navigates to the required links.
* Automatically detects and clicks buttons like Follow, Like, Share, etc.

## Installation & Usage

### Windows
1. Download the [main.py](https://github.com/anhnoine/n-tool_golike-tiktok/blob/main/main.py) file.
2. Run the script using Python:
```
python main.py
```

### Ubuntu / Debian / Mint / Termux
1. Download the script:
```
wget https://raw.githubusercontent.com/anhnoine/n-tool_golike-tiktok/refs/heads/main/main.py
```
2. Run the script using Python:
```
python main.py
```

Good luck!
